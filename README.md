## Architecture

The application is structured as a frontend/backend split with a relational database, running in containers.

- **Frontend**: React with TypeScript
- **Backend**: Python 3 (latest stable version)
- **Database**: PostgreSQL
- **Containers**: Podman (development only)

## Prerequisites

- [Podman](https://podman.io/) installed and running
- Node.js (for frontend development)
- Python 3 (for backend development)
